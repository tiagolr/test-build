To install Rippler copy the following files:

  * Rippler.vst3 -> C:\Program Files\Common Files\VST3
  * Factory -> C:\Users\tiago\AppData\Roaming\Rippler\presets
  * mallets -> C:\Users\tiago\AppData\Roaming\Rippler

The factory presets and audio mallets are not required to run the plug-in.
The folders should be copied with the folder name included so the resulting folders should be:

..\Common Files\VST3\Rippler.vst3
..\Rippler\presets\Factory
..\Rippler\mallets

Note that Rippler folder will not exist until you start Rippler for the first time.
Either run it a first time and then move the contents or create the Rippler directory first.
